# notthatcooked

A Spanish finals study game with XP, player profiles, and a Supabase-powered cross-device leaderboard.

## Deploy

This is a static site. Upload `index.html` to a GitHub repository and enable GitHub Pages from the repository's `main` branch.

Upload the `assets/` folder too. It contains the Raven Fantasy Icons spritesheet, Tiny RPG Soldier/Orc sprites, woods tiles, and the Cute Fantasy tiles/characters now used by the overworld and battle scenes.

Note: the Cute Fantasy free pack readme says it can be used for non-commercial projects, but cannot be redistributed or resold as an asset pack.
